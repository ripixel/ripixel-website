This website! It's cool, isn't it? Why use a static site generator when you can just... y'know... write the HTML yourself?

In the end, I decided to go as simple-as-possible. This site shouldn't be all-singing-and-dancing, because it's not what's needed. What's needed is something super simple, clean, and most importanly quick. So, it's pure HTML and CSS, with a really simple template copy/paste script so I don't have to write the nav/header/footer out everytime. Neat!
