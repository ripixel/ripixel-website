FitGlue

An automation platform for fitness enthusiasts who are tired of manually copying their workouts between apps. Ever finished a killer workout on one app and then had to faff about re-entering it all somewhere else? FitGlue connects your fitness services together and automatically syncs activities — with AI-generated descriptions, parkrun result lookups, and more.

Built on a React frontend with a Go and TypeScript backend, all running on Google Cloud.

Skier

A custom static site generator — because the world definitely needed another one of those. A task-based build system that transforms JSON and Markdown into static HTML. Handles pagination, RSS feeds, SEO sitemaps, and more. Powers this very website, the FitGlue marketing pages, and the Bingham Sunday Running Club site.

docs → GitHub →

Bingham Sunday Running Club

A full digital ecosystem for a small town running club that probably didn't need this level of engineering. React-based race tracker app, Skier-powered club website, and an automated CI/CD pipeline. Every Sunday morning results are live before the slowest runner finishes. Features automatic PB tracking, a runners directory, Spotify integration, and weather data enrichment.

visit the club → GitHub →

Presentation Timer

A presentation countdown timer with background music streaming — because every stream could use some suspense. Having the countdown and music in the same browser tab means streaming tools can pick up the audio cleanly, without routing your entire system audio.
